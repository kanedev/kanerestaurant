## 🚀 Install :

```sh
$ git clone https://github.com/kanedev/kanerestaurant
$ cd kanerestaurant
$ # Go to : kanerestaurant/src/shared/baseUrl.js and put your adress IP
$ json-server --host YOUR_ADRESS_IP_HERE  --watch db.json -p 3001 -d 2000
$ yarn 
$ yarn start 
```
